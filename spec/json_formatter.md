# JSON格式化工具网页 (json_formatter)

## 概述
一个独立的单文件HTML页面，提供JSON/纯文本的格式化、压缩、转义处理等功能。

## 页面布局
1. 页面分为左右两部分：
   - 左侧：文本输入框（textarea），用于输入JSON文本或纯文本。
   - 右侧：输出区域，显示经过处理后的文本，支持JSON语法高亮。
2. 输入框和输出框上方或中间放置功能按钮栏。

## 功能按钮
1. **格式化JSON**: 将输入框中的JSON文本进行美化格式化（缩进显示）。若输入不是合法JSON，则尝试对不完整JSON（如某个完整JSON的前半段）进行token级格式化；若连token也无法识别，则提示错误信息。
2. **压缩单行**: 将输入框中的JSON文本压缩为单行（去除多余空白和换行）。若输入不是合法JSON，则在输出区域提示错误信息。
3. **去除转义**: 将输入框中的文本进行反转义处理（如 `\"` → `"`，`\\n` → 换行符等），直接输出反转义后的文本。
4. **去除转义+格式化**: 同"去除转义"，但处理后如果是合法JSON则自动格式化显示。
5. **增加转义**: 将输入框中的文本进行转义处理（如 `"` → `\"`，换行符 → `\\n` 等），输出为可嵌入字符串的转义文本。

### 辅助按钮
- 输入框右下角提供"粘贴"按钮，点击后从剪贴板读取文本填入输入框。

## JSON语法高亮
- 输出区域对JSON内容进行语法高亮渲染（使用HTML元素而非纯文本叠加）：
  - 字符串值：深黄色
  - 布尔值/数字值：粉色
  - null：红色
  - 键名：浅蓝色
  - 符号（括号、冒号、逗号）：默认色

### 折行开关
- 输入框和输出框各自提供"折行"开关，可切换文本是否自动换行显示。

### JSON折叠
- 输出区域的JSON对象（`{}`）和数组（`[]`）支持点击折叠/展开，折叠后显示摘要信息（如键数量或元素数量）。

### 复制按钮
- 输出区域右下角提供"复制"按钮，点击后将输出的纯文本内容复制到剪贴板。

## 技术约束
- 单文件HTML，内嵌CSS + JavaScript，无任何外部依赖。
- 文件放置于项目根目录下html文件夹，文件名为 `json_formatter.html`。
